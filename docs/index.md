# Início

## 1. Introdução

Essa página destina-se ao armazenamento da documentação do projeto "TwoDo", realizado na disciplina "Métodos de desenvolvimento de software", do semestre 2023/1. O objetivo do projeto é desenvolver uma plataforma onde os usuários terão acesso a fóruns de discussão e grupos de interesse, que permitirão a interação com pessoas de diferentes áreas e a aquisição de conhecimentos relevantes para o desenvolvimento de suas carreiras.
